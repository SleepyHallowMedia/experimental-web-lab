About

This is a static, GitHub Pages-friendly experimental site. It intentionally uses:

  • Web Components + Shadow DOM
  • Popover API (with dialog fallback) — documented by MDN [2](https://developer.mozilla.org/en-US/docs/Web/API/Popover_API/Using)[3](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Global_attributes/popover)
  • CSS Anchor Positioning (with JS fallback) — documented by MDN [4](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Anchor_positioning)
  • Canvas demos and Web Audio
  • Service worker caching

If you want to expand: add more custom elements, new demo pages, or bring in optional libraries via CDN.
