CSS Anchor Positioning (progressive enhancement)

Anchor positioning tethers one element to another via CSS. MDN explains the module and properties. [4](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Anchor_positioning)

This repo uses anchor positioning when available; otherwise it falls back to getBoundingClientRect() placement.
